Events

Event system for monitoring and integration.

Overview

Guardrail emits events throughout the request evaluation process. You can listen to these events for logging, monitoring, and custom integrations.

typescript 
gr.on("decision.denied", (event) => {
  console.log("Request denied:", event.decision);
});

Available Events

Event Description
"rule.evaluate" Fired when a rule starts evaluation
"rule.allow" Fired when a rule allows a request
"rule.deny" Fired when a rule denies a request
"decision.allowed" Fired when a request is allowed
"decision.denied" Fired when a request is denied
"storage.error" Fired when storage operations fail
"ip-lookup.error" Fired when IP lookup fails
