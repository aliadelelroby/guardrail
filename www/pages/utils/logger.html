Logger

Logging utilities for Guardrail.

Overview

Guardrail includes a built-in logger that provides colored console output. Logging is enabled when debug: true is set in the configuration.

typescript 
const gr = guardrail({
  rules: [/* ... */],
  debug: true, // Enable logging
});

Log Levels

  • debug — Detailed debugging information
  • info — Informational messages
  • warn — Warning messages
  • error — Error messages

Custom Logger

You can provide a custom logger implementation:

typescript 
import type { Logger } from "@guardrail-dev/core";

const customLogger: Logger = {
  debug(message, ...args) {
    // Your logging implementation
  },
  info(message, ...args) {
    // Your logging implementation
  },
  warn(message, ...args) {
    // Your logging implementation
  },
  error(message, ...args) {
    // Your logging implementation
  },
};
