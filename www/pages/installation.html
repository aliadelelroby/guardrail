Installation

Install Guardrail using npm or your preferred package manager:

bash 
npm install @guardrail-dev/core

With Redis Support

For distributed deployments with Redis storage:

bash 
npm install @guardrail-dev/core ioredis

Using Yarn

bash 
yarn add @guardrail-dev/core

Using pnpm

bash 
pnpm add @guardrail-dev/core

Framework-Specific Packages

Guardrail works with all major Node.js frameworks. No additional packages are required for:

  • Express.js
  • Next.js
  • Nest.js
  • Fastify
  • Koa

All adapters are included in the main package.
